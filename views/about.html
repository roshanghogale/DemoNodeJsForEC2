About This Project

Created by: Roshan Ghogale

This Node.js application demonstrates EC2 AutoScaling by deploying on AWS infrastructure. I'm following a complete workflow from EC2 launch to AMI creation, Target Groups, and Application Load Balancer setup.

Key Components

Launch Templates

Define instance configuration including AMI, instance type, security groups, and user data

Auto Scaling Groups

Logical grouping of instances for scaling and management purposes

Scaling Policies

Rules that define when and how to scale your instances up or down

My Deployment Architecture

  • EC2 Instance: Running this Node.js app with proper configuration
  • Custom AMI: Captured from configured EC2 instance
  • Target Group: Health checks and instance registration
  • Application Load Balancer: Traffic distribution and SSL termination
  • Auto Scaling Group: Dynamic scaling based on metrics
